BAB 1

JENIS-JENIS

Ada 4 jenis arsitektur komputer :
1. SISD = Singel Instructian Single Data
Merupakan hasil dari rancangan Van Neumann. Komputer ini jenis SISD hanya memiliki satu jenis processor saja.
2. SIMD = Single Instruction Multiple Data
SIMD berbeda dari SISD jenis ini memiliki jumlah processor dengan instruction sama.
3. MISD = Multiple Instruction Singel Data
Memiliki banyak processor dengan instruksi berbeda tetapi pengolahan datanya sama.
4. MIMD = Multiple Instruction Multiple Data
Memiliki banyak processor dengan Instruksi serta pengolahan data yang berbeda.

BAB 1

FUNGSI

Berfungsi sebagai perancang awal komputer dengan menggunakan bahasa programan.
Arsitektur komputer masih memiliki beberapa fungsi lainnya :
1. Perencanaan Awal Komputer
Komputer disusun dan dirancang dengan baik mungkn agar kinerjanya maksimal perancangan ini juga berfungsi untuk mengetahu apa saja yang kurang atau perlu diperbaiki.
2. Mengontrol Komputer Yang Ada Pada Komputer
Bertujuan agar kinerjanya komputer bisa berjalan dengan baik / maksimal. Fungsi dari kontrol ini juga bisa membantu pengguna untuk bisa menjalankan banyak pekerjaan atau apliksi dalam satu komputer.
3. Membantu Menentukan Aplikasi Atau Programan Yang Akan Digunakan
Arsitektur komputer dapat membantu programmer dalam menentukan aplikasi / program apa yang cocok dengan komputer sehingga bisa disesuiakan dengan kebutuhan pengguna dan berjalan dengan maksimal.

CPU

PENGERTIAN
CPU merupakan singkatan dari Central Prosessor Unit yang sering diartikan oleh manusia sebagai tubuh maupun dari otak sikomputer. CPU juga dapat mengolah data-data yang telah masuk kedalam komputer.

BAB 2

KOMPONEN CPU

- Control Unit : Mengendalikan operasi dasar CPU dan mengarahkan aliran data dan instruksi dalam prosesor.
- Arithmetic and Logic Unit (ALU) : Untuk melakukan operasi dasa aritmetika dan operasi lohika berdasarkan instruksi yang ditentukan.
- Register : Penyimpanan sementara yang digunakan untuk menyimpan data yang sedang diproses atau hasil perhitungan sementara selama eksekusi instruksi komputer.
CPU Interconnection : Sistem koneksi dan bus yang menghubungkan komponen internal CPU dengan bus-bus eksternal.

Komponen penting dalam unit pemrosesan pusat(CPU) pada komputer. ALU bertanggung jawab untuk melakukan operasi aritmatika (seperti penjumlahan, pengurangan, perkalian dan pembagian.)

BAB 2

JENIS-JENIS REGISTER

- MAR (Memory Addres Register)
Register ini digunakan untuk mengakses data dan instruksi dari memori selama fase eksekusi suatu instruksi.
- MDR (Memory Data Register)
Register unit kontrol komputer yang berisikan data yang akan disimpan di penyimpanan computer(misal RAM).
- MBR (Memory Buffer Register)
Untuk menyimpan berbagai jenis instruksi komputer dan data yang dapat ditransfer antar memory komputer.

BAB 4

JENIS-JENIS MEMORY

- Memory Read Only (ROM)
Tipe memorori yang berisikan data yang tidak dapat diubah oleh pengguna data dalam ROM tidak dapat diubah oleh pengguna.
- Memory Read / Write
Tipe memori yang memungkinkan pengurus untuk membaca dan menulis (menyimpan atau mengubah data ke dalam memori) sesuai kebutuhan.
- Static Memory
Tipe memori yang dapat menyimpan data tanpa memerlukan daya listrik yang berkelanjutan dan datayang disimpan bersiffat tetap hingga diubah atau dihapus secara sengaja.
- Dynamic Memory
Tipe memori yang digunakan untuk menyimpan data yang dapat di baca dan ditulis oleh perangkat lunak komputer.
